My Social

A Social Networking Site

Project Overview

My Social is a Facebook-like social networking site built using only HTML, CSS, and JavaScript. It demonstrates core social media functionality including news feed, profiles, likes, comments, and sharing without requiring any backend server or database.

How It Works

This project is built entirely with frontend technologies. All data is stored in JavaScript arrays that simulate a database. User interactions like posting, liking, and commenting all update these arrays in real-time to provide a dynamic experience.

For a real social network, this would be connected to a backend server and database, but this demo focuses on the frontend user experience and interactions.

Technologies Used

  • HTML5 - Structure and content
  • CSS3 - Styling and responsive design
  • JavaScript - Client-side functionality and DOM manipulation
  • Font Awesome - Icons for the user interface

Note: This is a demo project to showcase frontend development skills. In a production environment, a social network would require backend services, databases, and proper security measures. All data in this demo is reset when the page is refreshed.
